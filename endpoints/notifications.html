# Notifications-Endpoint

Endpoint to Fetch all latest notifications.

GET /notifications

# Usage

  1. Go here:
https://results-restapi.up.railway.app/notifications
1

# Example

Copy and paste this url in a new tab:

https://results-restapi.up.railway.app/notifications
1
