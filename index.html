SGPA-REST-API

# Introduction

This has been specifically built to suppliment the mobile application that I have developed. Feel free to use this irrespective of the mobile-app, with which you can integrate into your own applications/projects to get the data. This RestAPI provides the results for all regulations, and of all types ( regular and supplementary ).

# Why I built this RestAPI?

One day, I had to calculate the SGPA for each semester at college (manually), so I had an idea to develop a Cross Platform Appopen in new window. I verified if there was a public API access from our Universityopen in new window so that I can fetch the details of students and their respective results, but, to my suprise - there is none.

So, I just created my own.
